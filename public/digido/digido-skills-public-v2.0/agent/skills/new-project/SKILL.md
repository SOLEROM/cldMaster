# הקמת פרויקט חדש

## מתי להשתמש
כאשר הארכיטקט אומר "אני רוצה להתחיל פרויקט חדש" או "בוא נקים משהו חדש".

## התהליך (שיתופי)

### שלב 1: הבנת הצורך
שאל את הארכיטקט:
- מה הפרויקט אמור לעשות?
- למי זה מיועד?
- האם יש דוגמה דומה שראית?

### שלב 2: יצירת Specification
הפעל את ה-**specifications** skill:
```
/specifications
```

הוא ייצור spec.md מסודר בתיקיית .digido/

### שלב 3: המלצה על Workflow
הפעל את ה-**skill-advisor** כדי לייעץ לארכיטקט:
```
/skill-advisor
```

הוא יגיד לארכיטקט אם לעבוד עם:
- `/python-professional-setup` (אם Python)
- `/plan` + `/execute` (אם פרויקט מתוכנן)
- `/map` (אם יש codebase קיים)

### סיום
הראה לארכיטקט:
1. איפה נמצא ה-spec.md
2. מה הצעד הבא לפי skill-advisor
3. שאל: "רוצה שאתחיל?"
