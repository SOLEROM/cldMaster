# Context Health Monitor

## 🎯 מטרה

לשבור מעגלים של debugging כושל, לזהות מתי אנחנו תקועים, ולהחליט ביחד מה הלאה.

---

## 🔧 מתי להפעיל

**המשתמש מפעיל** כש:
- אותה שגיאה חוזרת 3+ פעמים
- מרגיש שאנחנו הולכים במעגלים
- לא בטוח אם אנחנו מתקדמים
- רוצה "לעצור ולחשוב" לפני שממשיכים

**הסוכן מציע** כש:
- זיהה שניסה אותה גישה פעמיים
- נכשל 3 פעמים על אותו דבר
- לא בטוח מה הבעיה האמיתית

---

## 📋 מה קורה כשמפעילים

### שלב 1: סיכום המצב (הסוכן)

```
🔍 **איפה אנחנו:**
- מה ניסינו עד כה (רשימה קצרה)
- מה עבד, מה לא עבד
- מה השגיאה/בעיה האחרונה

💡 **מה אני חושב:**
- ההשערה שלי למקור הבעיה
- למה הגישות הקודמות נכשלו

📌 **קבצים מעורבים:**
- [file.py:42] - מה המצב שלו
```

### שלב 2: שיחה משותפת

הסוכן שואל:
```
🤔 **מה עושים עכשיו?**

אפשרות 1: לנסות גישה שונה לגמרי - [הסבר]
אפשרות 2: לחפש מידע חיצוני (דוקומנטציה/Stack Overflow)
אפשרות 3: להפסיק ולהתחיל session חדש עם `/pause`
אפשרות 4: אתה רואה משהו שאני לא?
```

### שלב 3: החלטה (משתמש)

המשתמש בוחר, והסוכן ממשיך לפי ההחלטה.

---

## ✅ כללי התנהגות

### כלל 3 הניסיונות

**אם אותה שגיאה נכשלה 3 פעמים:**
1. ⛔ **STOP** - לא לנסות fix רביעי
2. 💬 **הצע** את ה-skill הזה למשתמש
3. 🚫 **אל תמשיך** בעיוור

### כלל המעגליות

**אם אני רואה שחזרתי על אותה גישה:**
1. 🔁 אודה: "אני רואה שניסיתי את זה כבר"
2. 📝 אסכם מה ניסיתי
3. 💡 אציע גישה שונה לגמרי
4. 🛑 או אמליץ על ה-skill הזה

### כלל אי-הוודאות

**אם אני לא בטוח:**
1. 🗣️ אגיד זאת בגלוי: "אני לא בטוח אם..."
2. ❓ אשאל את המשתמש במקום לנחש
3. 🎯 אציע את ה-skill הזה לדיון משותף

## 🎭 דוגמה לשיחה

**משתמש:** `/context-health`

**סוכן:**
```
🔍 **איפה אנחנו:**

ניסיתי לתקן את שגיאת ה-import ב-main.py 3 פעמים:
1. ✗ שיניתי את הנתיב ב-sys.path - קיבלתי ModuleNotFoundError
2. ✗ הוספתי __init__.py - עדיין אותה שגיאה
3. ✗ שיניתי את שם המודול - עכשיו יש ImportError חדש

💡 **מה אני חושב:**
הבעיה היא לא בנתיב - אני חושב שיש circular import בין utils.py ל-main.py

📌 **קבצים מעורבים:**
- main.py:15 - מנסה לייבא את calculate
- utils.py:3 - מייבא משהו מ-main.py

🤔 **מה עושים עכשיו?**

אפשרות 1: נבדוק circular imports עם import-graph tool
אפשרות 2: נעביר את calculate למודול שלישי (no_deps.py)
אפשרות 3: נתחיל session חדש - אולי החמצנו משהו
אפשרות 4: משהו אחר שאתה רואה?
```

**משתמש:** "בוא ננסה אפשרות 2"

**סוכן:** "מעולה, אני יוצר..."

---

## 🔗 קשרים

- משלים את **`/pause`** - אם מחליטים לעצור
- משלים את **`/debug`** - כשצריך debugging שיטתי
- **לא יוצר קבצים** - הכל בשיחה
