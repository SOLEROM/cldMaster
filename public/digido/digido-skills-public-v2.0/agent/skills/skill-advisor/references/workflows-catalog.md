# קטלוג Workflows (תהליכי עבודה) — מידע מלא

## ליבה — הזרימה הראשית
| פקודה | מה עושה | מתי |
|-------|---------|-----|
| `/new-project` | אתחול פרויקט: שאלות → מחקר → דרישות → roadmap | פרויקט חדש |
| `/discuss-phase N` | דיון על שלב לפני תכנון — הבנת העדפות | לפני /plan |
| `/plan N` | מחקר + תכנון + אימות לשלב | אחרי discuss |
| `/execute N` | ביצוע כל התוכניות בגלים מקבילים | אחרי plan |
| `/verify N` | בדיקה ידנית שהעבודה עובדת | אחרי execute |
| `/complete-milestone` | סגירת אבן דרך, תיוג release | כשכל השלבים הושלמו |
| `/new-milestone` | התחלת גרסה חדשה | אחרי complete |
