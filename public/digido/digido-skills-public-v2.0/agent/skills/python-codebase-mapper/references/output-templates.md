## Special Cases

### Empty / New Project
```
ARCHITECTURE.md:
  Overview: "New project — structure not yet established"
  Components: "None yet"
  Recommendations: Suggest structure based on project type

STACK.md:
  Fill in what's known (Python version, any initial dependencies)
  Mark unknowns as "TBD"
```

### Monorepo / Multi-Package
```
Map each package separately, then document relationships:
  packages/
    auth/      → ARCHITECTURE section for auth
    api/       → ARCHITECTURE section for api
    shared/    → ARCHITECTURE section for shared

  Cross-package dependencies:
    api → imports from shared
    auth → imports from shared
```

### Windows-Specific Notes
```
- Paths use backslash: D:\project\src\module.py
- venv activation: .\venv\Scripts\activate (not bin/activate)
- PowerShell execution policy may block scripts
- Long paths (>260 chars) can cause issues
- .env files: check line endings (CRLF vs LF)
- File locking: Windows locks open files more aggressively
```
