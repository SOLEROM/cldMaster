# Error Fix Phase Workflow

## שלב 1: מצא את התכנית הפעילה
// turbo
רשום את הקבצים בתיקייה `.digido/error-handling/plans/`

## שלב 2: קרא את התכנית
קרא את קובץ התכנית הפעילה (`*-fix-plan.md`)

## שלב 3: זהה פאזה נוכחית
מצא בטבלת המעקב את הפאזה הראשונה עם ⬜

## שלב 4: בצע את התיקון
// turbo
בצע את המשימות המפורטות בפאזה

## שלב 5: עדכן סטטוס
עדכן את הטבלה: ⬜ → ✅

## שלב 6: צור סיכום
// turbo
צור קובץ `.digido/error-handling/plans/[error-name]-phase{N}-summary.md`

## שלב 7: דווח
