# Focus AI — 50 Claude Code Skills Bundle

50 professional skills for Claude Code, organized by category.
Each skill is invoked with `/skill-name` inside Claude Code.

---

## Quick Install

**macOS / Linux:**
```bash
bash install.sh
```

**Windows (PowerShell):**
```powershell
.\install.ps1
```

The installer copies all skills to `~/.claude/skills/` in the correct format.

## Usage

After installation, open any project folder and run `claude`. Then:

```
> /code-review
> /write-tests
> /security-audit
> /commit-message
```

Each skill guides Claude through a structured, multi-step process.
