# Sales Script — תסריט מכירה ומסגרת שיחת גילוי

שיחות מכירה שנשמעות אנושיות ולא כתסריט — אבל מובנות מספיק כדי להגיע לתוצאה. כולל Discovery, Demo והתמודדות עם התנגדויות.

## What to Provide
- מוצר/שירות ו-Value Proposition ראשי
- פרופיל הלקוח האידיאלי (תפקיד, ענף, כאב ראשי)
- ממוצע דיל (ACV) ואורך מחזור מכירה
- שלב הצורך: Discovery Call / Demo / Closing / כולם
- 3–5 התנגדויות נפוצות שאתם שומעים
- מה מבדיל אתכם ממתחרים

## How Claude Works Through This
1. **Call Opening** — פתיחה שבונה אמון ומגדירה ציפיות ברורות
2. **Discovery Questions** — 8–12 שאלות שמגלות כאב, דחיפות ותקציב (SPIN / MEDDIC)
3. **Value Demonstration** — איך לחבר מה שגילינו למה שאנחנו מציעים
4. **Objection Handling** — תשובות מוכנות לכל התנגדות (Feel-Felt-Found + Reframe)
5. **Closing Framework** — איך לסגור ב-3 שיטות: Direct / Summary / Trial Close

## Output Format
**Sales Script Package:**
- Discovery Call Script (פתיחה + שאלות + סגירה לשלב הבא)
- Demo Framework (סדר הצגה לפי כאב לקוח, לא לפי פיצ'רים)
- Objection Handling Card (כל התנגדות + 2 תשובות)
- Email Templates: Pre-call / Follow-up / No-response
- Scoring Sheet — איך להעריך לקוח בסוף שיחה
