# Focus AI — 50 Claude Code Skills for Managers & Entrepreneurs

50 professional management skills for Claude Code, organized by category.
Each skill is invoked with `/skill-name` inside Claude Code.

---

## Quick Install

**macOS / Linux:**
```bash
bash install.sh
```

**Windows (PowerShell):**
```powershell
.\install.ps1
```
