# /specifications workflow

<objective>
Create comprehensive specifications for new or ongoing projects through collaborative investigation and vision clarification
</objective>

<when_to_use>
- Starting a new project and need to define requirements
- Ongoing project that needs task breakdown into milestones/phases
- Need to clarify vision and create technical spec
</when_to_use>

<process>
Read .agent\skills\specifications\SKILL.md and follow the process described there.

Work collaboratively with the architect to:
1. Investigate and clarify the vision
2. Ask the right questions based on project type
3. Research technical feasibility and updated stack (use context7 if needed)
4. Create spec.md in .digido/specifications/ folder

Present all communication in clear Hebrew.
</process>
