## How to Update

After every significant work session, add entries under `[Unreleased]`:

### Added
- New features, files, or capabilities

### Changed
- Modifications to existing functionality

### Fixed
- Bug fixes and corrections

### Removed
- Deprecated features or files

---

## Version Release Template

When releasing a version, move [Unreleased] items to:

## [X.Y.Z] - YYYY-MM-DD

### Added
- Feature list

### Changed
- Change list

### Fixed
- Fix list

### Removed
- Removal list

Then clear [Unreleased] for future changes.
