# תבניות מוכנות — העתק-הדבק למשתמש

## תבנית: פרויקט חדש מאפס
```
מה קורה: מתחילים פרויקט חדש מאפס.
הסבר: הסוכן ישאל אותך שאלות על מה שאתה רוצה לבנות,
   יחקור את הנושא, ויבנה תוכנית עבודה. אחר כך תבנו שלב-שלב.
אמור לסוכן הקוד: /new-project
   ואז עקוב אחרי השאלות שלו.
   כשהתוכנית מוכנה: /discuss-phase 1 → /plan 1 → /execute 1 → /verify 1
```

## תבנית: פרויקט Python חדש
```
מה קורה: בונים פרויקט Python חדש.
הסבר: הסוכן יקים סביבה וירטואלית, יתקין ספריות,
   ויבנה את המבנה הנכון לפרויקט.
אמור לסוכן הקוד: /new-project
   כשהתוכנית מוכנה: /python-build-session
```

## תבנית: הוספה לפרויקט קיים
```
מה קורה: מוסיפים פיצ'ר חדש לפרויקט שכבר קיים.
הסבר: הסוכן קודם ימפה את הקוד הקיים כדי להבין מה יש,
   ואז יתכנן את התוספת בלי לשבור את מה שעובד.
אמור לסוכן הקוד:
   1. /map (רק בפעם הראשונה שנכנסים לפרויקט)
   2. /new-milestone "שם הפיצ'ר"
   3. /plan 1 → /execute 1 → /verify 1
```

## תבנית: תיקון באג
```
מה קורה: משהו שבור וצריך לתקן.
הסבר: הסוכן ינתח את הבעיה בצורה שיטתית —
   ינסה פתרונות ויוודא שהתיקון באמת עובד.
אמור לסוכן הקוד: /debug "תאר את הבעיה כאן"
   לפרויקט Python: /python-verify-and-fix
```

## תבנית: משימה קטנה
```
מה קורה: צריך לעשות דבר קטן וממוקד.
הסבר: אין צורך בתכנון מלא. פשוט תגיד לסוכן מה אתה רוצה.
אמור לסוכן הקוד: תתאר את המשימה ישירות.
```

## תבנית: חזרה לעבודה
```
מה קורה: חוזרים לפרויקט שעבדנו עליו בסשן קודם.
הסבר: הסוכן יטען את כל המידע מהסשן הקודם
   ויראה לך איפה עצרת ומה הצעד הבא.
אמור לסוכן הקוד: /resume
   ואז: /progress (לראות מיקום נוכחי)
תזכורת: תמיד תסיים סשן עם /pause!
```

## תבנית: מצגת / מסמך
```
מה קורה: יצירת מסמך או מצגת מעוצבים.
הסבר: הסוכן ייצור קובץ HTML יפה שאפשר לפתוח בדפדפן.
אמור לסוכן הקוד:
   למצגת: "צור מצגת HTML בנושא [X] עם צבעים [Y]"
   למסמך Digimate: "צור הצעת מחיר / סיכום בסגנון Digimate"
```

## תבנית: חיבור ל-Firebase
```
מה קורה: מחברים אפליקציה ל-Firebase.
הסבר: זה פרוטוקול מודרך — הסוכן ילווה אותך צעד אחרי צעד,
   כולל מה לעשות בקונסול של Firebase ומה הוא עושה בקוד.
אמור לסוכן הקוד: "חבר את הפרויקט ל-Firebase"
   הסוכן ישתמש ב-firebase-app-deployment-protocol אוטומטית.
```
