# superpowers

* https://github.com/obra/superpowers
* https://claude.com/plugins/superpowers

## about
* steps back and asks you what you're really trying to do.
* teased a spec out of the conversation
* signed off on the design
* puts together an implementation plan
* launches a subagent-driven-development process, having agents work through each engineering task, inspecting and reviewing their work

# Installation

```
/plugin install superpowers@claude-plugins-official
```
